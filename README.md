## ⚙️ Настройка переменных окружения

Для установки зависимостей необходимо настроить авторизацию в реестре GitHub Packages через переменную `NODE_AUTH_TOKEN`.

### Зачем это нужно?
Файл .npmrc настроен на использование внешней переменной. Это исключает хранение секретных ключей в коде и позволяет каждому разработчику использовать свои права доступа.

### Инструкция по установке токена

Вам потребуется Personal Access Token (PAT) с разрешением `read:packages`.

#### 1. Windows (PowerShell)
Выполните команду и перезапустите терминал:
```
setx NODE_AUTH_TOKEN "ваш_токен"
```

#### 2. macOS (zsh)
Добавьте строку в конфиг профиля:
```
echo 'export NODE_AUTH_TOKEN=ваш_токен' >> ~/.zshrc
```

#### 3. Linux (bash)
Добавьте строку в конфиг профиля:
```
echo 'export NODE_AUTH_TOKEN=ваш_токен' >> ~/.bashrc
```

ВАЖНО: После внесения изменений обязательно перезагрузите терминал или IDE, чтобы новые настройки вступили в силу.
